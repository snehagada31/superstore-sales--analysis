# Superstore Sales Data Analysis

## Project Overview
This project performs Exploratory Data Analysis (EDA) on a retail sales dataset to understand sales trends, regional performance, and top-selling products.

The analysis was performed using Python in Google Colab.

## Tools Used
Python  
Pandas  
NumPy  
Matplotlib  

## Dataset Information
The dataset contains sales transactions of a retail superstore including:

Order Date  
Region  
Category  
Product Name  
Sales  

## Analysis Performed
- Data cleaning and preprocessing
- Converting order date to datetime format
- Creating a Month column for time analysis
- Sales analysis by region
- Sales analysis by category
- Monthly sales trend visualization
- Top 10 products by sales
- Correlation analysis between Month and Sales

## Key Insights
- The West region generated the highest total sales, with East region close behind.
- Technology category recorded the highest revenue among all categories.
- The top selling product is Canon imageCLASS Advanced Copier.
- Sales peak during November while February records the lowest sales.
